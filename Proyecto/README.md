# Proyecto

## Descripcion

Este proyecto consiste en una pagina web desarrollado como proyecto final de la materia de programacion para internet.

## Objetivo

Desarrollar un sistema que permita visualizar, registrar, editar y eliminar usuarios registrados en una base de datos.

El usuario no podra navegar por la pagina si no inicia sesion, este tendra que ser redirigido al login y de igual manera si tiene una sesion iniciada no podra ir al login son antes cerrar la sesion actual.

## Caracteristicas

- Lista de empleados
- Eliminar empleados
- Alta de empleados
- Ver detalles de empleado
- Edicion de empleados
- Adjuntar archivo
- Login
- Autentificación y validación con sesiones
- Menu
- Cerrar sesion

## Tecnologias

- HTML
- JavaScript
- PHP
- CSS
- MySQL
